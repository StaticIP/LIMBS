HOTOCO Emotional Transfer Mechanisms

The section Communication Rules describes how to get your HOTCO Agent to send messages, However, these rules do not describe how the agent receives these messages. Well, the receiver can receive messages via a range of emotional transfer mechanisms. These mechanisms are:
  1. Emotional Contagion
  2. Emotional Altruism
  3. Emotional Empathy
All of these mechanisms rely on the transferring of emotional vectors along with a certain set of propositions. Contagion transfers emotional information regarding actions. Altruism transfers emotional information regarding goals. Empathy transfers emotional information regarding evidence. These allow for common socio-emotional phenomena to occur through the spread of emotions as well as information in communication.
To access these mechanisms, once again, you must go to the HOTCO interface. Select “Emotional Transfer Rules” from the “Agent Options” menu at the top of the interface. This will open a dialog showing four text boxes. The first list, “Agents,” will include all the agents that one can communication with, as well as a DEFAULT person. To use emotional transfer mechanisms, just select an item from the “Agents” list, then click on the “Add” button below one of the other “Contagion,” Altruism,” and “Empathy” lists to add that agent to that list. Removing an agent works the same way. Just select the agent from the “Agents” list, and hit remove from below whatever list you want to remove the agent from.
